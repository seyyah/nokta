## Project Summary

Idea Card Splitter is a mobile application that helps users organize messy brainstorming notes into clean and structured idea cards. The app accepts raw text inputs such as chat exports, bullet notes, or brainstorming drafts. It removes duplicates, merges similar concepts, and creates readable idea cards.

The main purpose of the project is to save time during idea generation and help users focus on the best concepts instead of manually organizing notes.

## Features

- Paste raw brainstorming notes
- Detect duplicate ideas
- Merge similar concepts
- Generate idea cards
- Clean mobile user interface
- Android APK support
- AI-powered text structuring
