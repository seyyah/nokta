# NOKTA Spec

## Track

Track A — Idea Clarifier / Spec Generator

## Project Summary

This mobile app helps users turn rough ideas into structured one-page product specs by asking 3–5 engineering questions.

## Decision Log

- I chose Track A because it is clear and practical.
- I used Expo React Native for fast mobile prototyping.
- I kept text input as the main flow and added voice as a supporting feature.
- I used mock/local AI logic so the app works without a backend.
